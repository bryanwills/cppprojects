# HEX-DECODER
Hex Decoder is a program that can decode hexadecimal based on the given
command line arguments

# COMPILING AND EXECUTING
```sh
# compile
gcc hex-decoder.c -lm -o hex-decoder
# execute
./hex-decoder <hex>
```
